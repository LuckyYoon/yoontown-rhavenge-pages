// 01 — Big Idea

Face the Rhavenger
of Yoontown

Yoontown Rhavenge is a top-down bullet hell boss-rush game inspired by the manic shooter genre. You play as Gilbert, an unlikely hero armed with nothing but a slingshot, who must face the fearsome Rhavenger of Yoontown — a multi-phase boss that grows more ruthless with every hit it takes.

The core goal is pure survival: dodge an escalating storm of projectiles, find openings to fire back, and outlast the boss through both of its distinct phases. Every sprite, sound, and attack pattern was crafted from scratch for this game.

Player

Gilbert wields a slingshot and moves freely in 2D space. Precision and quick reflexes are your only weapons — one hit from a projectile and the run is over.

The Boss

The Rhavenger of Yoontown has PHASE 1PHASE 2. Push it into Phase 2 and it unlocks entirely new, harder attack patterns.

Genre

Bullet hell (manic shooter) — a subgenre where the challenge comes from reading and navigating dense, layered streams of enemy projectiles rather than raw combat power.

Tech

Built entirely in Python 3.14 using the community-edition pygame-ce library. Clean MVC-inspired architecture across three core modules.
// 02 — Unique Features

What Makes It Special

  • 💀 Two-Phase Boss Fight Once the Rhavenger reaches a health threshold it transforms — new attack patterns, faster projectiles, and a tougher second phase that demands even sharper reactions.
  • 🎵 Original Soundtrack All music and sound effects were composed from scratch specifically for Yoontown Rhavenge. Audio cues are woven into gameplay — headphones are recommended for the full experience.
  • 🎨 Hand-Crafted Sprites Every sprite and animation in the game was created exclusively for this project — no stock assets, all original pixel art.
  • 🌀 Bullet Hell Patterns Inspired by the manic shooter genre, the boss fires complex, patterned projectile arrangements that require reading and memorization to navigate.
  • 🖥️ Simple Controls WASD to move, Spacebar to fire. Zero barrier to entry — the challenge comes entirely from the game, not from learning controls.
  • 🧱 Clean Architecture Code is split across focused modules: ytr_classes.py for game objects, ytr_config.py for tunable constants, and a full unit-test suite.

// 04 — Installation

Get Up and Running

Running Python 3.14 is recommended. The game should work on earlier versions of Python 3, but switch to 3.14 if you encounter any issues.

  1. Install Python 3.14
    Download and install Python 3.14 from the official site. 
    https://www.python.org/downloads/release/python-3144/
  2. Clone the Repository
    git clone https://github.com/olincollege/yoontown-rhavenge
  3. Install pygame-ce
    pip install pygame-ce

    On Windows, you may need to prefix with py -m:

    py -m pip install pygame-ce
  4. Run the Game
    cd yoontown-rhavenge
python yoontown_rhavenge.py
